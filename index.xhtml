IRI/LDEO Climate Data Library IRI/LDEO Biblioteca de Datos Climáticos IRI/LDEO Datathèque du Climat МНИИ/ОЗЛД Библиотека Климатических Данных

The IRI Data Library is a powerful and freely accessible online data repository and analysis tool that allows a user to view, analyze, and download hundreds of terabytes of climate-related data through a standard web browser.

La Biblioteca de Datos del IRI es un poderoso almacén de datos disponible libremente en línea y una herramienta de análisis que permite visualizar, analizar, y bajar cientos de terabites de datos relacionados al clima, a través de un navegador estándar del internet.

La Datathèque de l'IRI consiste en un puissant répertoire de données libre d'accès en ligne et en un outil d'analyse qui permet la visualisation, l'analyse et le téléchargement de centaines de térabits de données liées au climat via un navigateur internet standard.

It is a powerful tool that offers the following capabilities at no cost to the user:

  • access any number of datasets;
  • create analyses of data ranging from simple averaging to more advanced EOF analyses using the Ingrid Data Analysis Language;
  • monitor present climate conditions with maps and analyses in the Maproom;
  • create visual representations of data, including animations;
  • download data in a variety of commonly-used formats, including GIS-compatible formats.

Es una herramienta poderosa que permite lo siguiente libre de costos :

  • acceder a una gran cantidad de datos;
  • crear analises de datos desde un promedio básico hasta analises más avanzados como la descomposición en las Funciones Empíricas Ortogonales al utilizar el Language de Análisis de Datos Ingrid;
  • monitorear a las condiciones climáticas actuales a través de mapas y analises en el Maproom;
  • crear representaciones visuales de los datos, incluyendo animaciones;
  • bajar datos en diferentes generalizados formatos, incluyendo formatos compatibles con los SIG.

Il s'agit d'un outil puissant qui offre les fonctionnalités suivantes librement:

  • accéder à un grand nombre de données;
  • faire de l'analyse de données depuis le calcul d'une simple moyenne jusqu'à des calculs plus avancés tels que la décomposition Orthogonale aux Valeurs Propres en utilisant le Langage d'Analyse de Données Ingrid;
  • observer les conditions climatiques actuelles à travers des cartes et autres analyses dans la Maproom;
  • créer des représentations visuelles des données, y compris des animations;
  • télécharger les données dans une grande variété de formats usuels, y compris des formats compatibles avec les SIG.
