Explorasi Tool Prakiraan untuk Aktivitas Kebakaran di Wilayah Kalimantan, Indonesia

Bagian ini menjelaskan prakiraan kemungkinan tinggi atau rendahnya aktivitas kebakaran hutan di Selatan, Tengah, Barat dan Timur Propinsi Kalimantan Indonesia. Penelitian ini telah dilakukan oleh IRI bekerjasama dengan CCROM SEAP Institut Pertanian Bogor (IPB) didukung oleh CARE Indonesia. Saat ini IRI dan Badan Meteorologi dan Geofisika (BMG) sedang membandingkan metode yang cocok untuk pendugaan kebakaran hutan.

Berdasarkan hasil monitoring indeks suhu muka laut (SST) di wilayah NINO4 dari bulan April sampai September, memungkinkan untuk melakukan pendugaan atau estimasi aktivitas kebakaran 1-2 bulan ke depan. Analisis ini berdasarkan data hotspot dari tahun 1998-2006 (Sumber dari NOAA-AVHRR) dan pada NINO4 index. Keempat grafik di bawah ini memperlihatkan hasil analisis untuk masing-masing provinsi di wilayah Kalimantan, Indonesia.

Indeks NINO4 dalam bentuk deret waktu (ditunjukkan oleh grafik) yang diperbaharui setiap bulan. Berdasarkan keterkaitan antara indeks NINO4 dengan aktivitas api (kebakaran), pengguna dapat memperoleh gambaran dan memprediksi aktivitas api 1-2 bulan kedepan, apakah akan mengalami peningkatan atau penurunan dari nilai rata-rata.

Menduga aktivitas api dua bulan lebih awal:


Grafik-grafik ini menunjukan hubungan antara jumlah kebakaran (titik panas) dengan indeks NINO4 dua bulan sebelumnya untuk empat provinsi di Kalimantan. Garis putus-putus menandai nilai median untuk NINO4 dan jumlah api (titik panas). Untuk menentukan tingkat risiko api dua bulan yang akan datang, gunakan nilai indek NINO4 bulan April, Mei, Juni, Juli atau Agustus, dan dapatkan jumlah titik api dugaan sesuai dengan nilai NINO4 terkait. Nilai-nilai indeks NINO4 yang lalu dan sekarang dapat ditemukan dari grafik di halaman atas.

Tabel kontigensi (angka-angka yang terkait dengan n, N, f, dan F pada garfik) menunjukkan banyaknya titik api di ke empat kuadran yang dideliniasi oleh garis putus-putus. Kontigensi dapat digunakan untuk mengidentifikasi peluang untuk mendapatkan jumlah titik api di atas atau di bawah nilai rata-rata sesuai dengan indeks NINO4 terkait.
